## 프로토타입
- JS에서 함수 생성 방법 3가지
  - 함수 선언문
  - 함수 표현식
  - Function() 생성자 함수

## 함수 리터럴
- JS에서는 함수도 일반 객체처럼 값 취급
- 함수 리터럴로 함수 생성 가능
- 함수 선언문이나 함수 표현식 모두 함수 리터럴 방식으로 함수 생성

```JAVASCRIPT
function add(x, y) {
  return x + y;
}

1) function 키워드로 시작
2) 함수명은 선택 사항. 함수명이 없는 경우는 익명함수라 칭함
3) 매개변수에 자료형 기입 X
```
